## Usage guidance

- “Platform Administration” is reserved for Central-only routes and login.
- “School Administration” is reserved for school-scoped routes and login.
- “Archive” refers to the verified archived-documents view only; no legal retention semantics are implied.
- “Effective Permission” should be used when the UI is showing computed permissions rather than role membership alone.
